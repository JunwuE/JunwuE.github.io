# Junwu E - Personal Website

Personal academic homepage built with vanilla HTML/CSS/JS, hosted on GitHub Pages.

## Pages

- **Home** - Interactive terminal + Bayer-dithered robot with speech bubbles
- **Profile** - Education, skills, research interests, awards
- **Publications** - Academic papers (CVPR 2026, RCAR 2025, etc.)
- **Projects** - Engineering and research projects
- **Contact** - Email, phone, GitHub, location

## Usage

```bash
git clone https://github.com/JunwuE/JunwuE.github.io.git
# Open index.html in browser, or:
python -m http.server 8000
```
