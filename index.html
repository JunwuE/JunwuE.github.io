Research Experience

Transformer and Trainable Bilateral Filter for Unsupervised Stereo Matching

2025.02 - 2025.06 | First Contributor

Supervisor: Prof. Rui Fan, MIAS Group, Tongji University

  • Proposed TBSMNet, an unsupervised stereo matching network integrating a hybrid Transformer-CNN encoder
  • Designed a ViT-CNN hybrid encoder combining global context modeling with local structure preservation
  • Introduced a trainable bilateral filter with adaptive parameters for noise suppression
  • Achieved significant accuracy improvements on SceneFlow and KITTI datasets

Stereo Vision-based Tactile Sensing

2024.5 - 2024.11

Supervisor: Prof. Rui Fan & Prof. Shaowei Cui (CASIA)

  • Implemented a vision-based tactile sensing system using binocular cameras
  • Built deep learning-based stereo processing framework for marker detection and disparity estimation
  • Conducted system calibration and parameter optimization for long-term stability
  • Developed real-time visualization and monitoring pipeline

Joint Learning of Monocular Depth Estimation and Semantic Segmentation

2024.10 - 2025.02 | First Contributor

Supervisor: Prof. Rui Fan, MIAS Group, Tongji University

  • Built multi-task joint learning framework with shared encoder and task-specific decoders
  • Designed multi-objective training strategy with adaptive loss balancing
  • Implemented 3D semantic point cloud reconstruction using Open3D
  • Demonstrated improvements in both segmentation accuracy and depth reconstruction

CarlaOcc: Instance-aware Occupancy Prediction Benchmark

2025.05 - Present | Second Contributor (with Yi Feng)

Supervisor: Prof. Rui Fan, MIAS Group, Tongji University

  • Collected multi-modal data from CARLA simulator for large-scale occupancy prediction dataset

Kitti360 Instances Dataset

Supervisor: Prof. Rui Fan, MIAS Group, Tongji University
